# Quick Start Guide 🚀

## Prerequisites
- Node.js 18+ installed
- npm package manager

## Installation & Setup (5 minutes)

### Step 1: Install Dependencies

**Backend:**
```bash
cd backend
npm install
```

**Frontend:**
```bash
cd frontend
npm install
```

### Step 2: Start the Servers

**Terminal 1 - Backend Server:**
```bash
cd backend
npm start
```
✅ You should see: `Server running on port 5000`

**Terminal 2 - Frontend Development Server:**
```bash
cd frontend
npm run dev
```
✅ You should see: `Local: http://localhost:5173/`

### Step 3: Open in Browser
Navigate to: **http://localhost:5173**

## Your First Resume (2 minutes)

1. **Personal Information**
   - Enter your name, email, phone, location
   - Add LinkedIn/GitHub (optional)

2. **Professional Summary**
   - Write a brief summary
   - Click "Optimize with AI" to enhance it

3. **Add Experience**
   - Click "Add Experience"
   - Fill in job details
   - Optimize descriptions with AI

4. **Add Education**
   - Click "Add Education"
   - Enter degree and institution

5. **Add Skills**
   - Type comma-separated skills
   - Example: JavaScript, React, Node.js, Python

6. **Choose Template**
   - Select from 4 professional designs
   - Preview updates instantly

7. **Download**
   - Click "Download as PDF"
   - Your resume is ready!

## Templates Overview

| Template | Best For | Style |
|----------|----------|-------|
| **Modern** | Tech, Startups, Creative | Vibrant colors, icons, modern layout |
| **Classic** | Finance, Law, Academic | Traditional, professional, text-focused |
| **Minimal** | Design, Consulting | Clean, elegant, typography-focused |
| **Creative** | Marketing, UX/UI | Sidebar layout, gradient styling |
